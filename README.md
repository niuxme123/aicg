# AICG 短视频剧本生成器

基于 OpenAI API 的短视频剧本智能生成系统，完整工作流：洗稿 → 排版 → 分镜 → 分镜优化 → 定妆照+场景 → 关键帧。

## 功能特点

- 🎭 **6步完整工作流**：从原始剧本到完整制作提示词
- 🤖 **基于 OpenAI GPT 模型**：支持 GPT-4o、GPT-4o Mini、GPT-3.5 Turbo
- 🎨 **美观的用户界面**：现代深色主题设计
- 📝 **实时进度显示**：清晰的状态指示器
- 💾 **本地存储**：API设置本地保存
- 📋 **复制和下载**：一键复制或下载结果
- 📱 **响应式设计**：适配桌面和移动设备

## 快速开始

### 1. 打开应用
直接双击 `index.html` 在浏览器中打开，或使用任意 HTTP 服务器。

### 2. 配置 API
1. 在右侧"API 设置"面板输入你的 OpenAI API Key
2. 选择 AI 模型（推荐 GPT-4o）
3. 点击"保存设置"

### 3. 测试连接
点击"测试连接"按钮验证 API 是否可用。

### 4. 输入剧本
在左侧输入框中粘贴原始剧本文本，或点击"加载示例"按钮获取示例。

### 5. 开始生成
- **一键生成**：点击"一键生成"执行完整6步流程
- **单步执行**：点击"单步执行"仅执行当前选中步骤

### 6. 查看结果
点击上方标签页查看各步骤生成结果：
- 洗稿结果 → 排版结果 → 分镜结果 → 优化分镜 → 定妆照 → 关键帧

### 7. 导出结果
使用右上角按钮复制所有结果或下载为文本文件。

## 工作流程详解

### 第1步：洗稿
重写原始剧本，更换人物名称，重构台词和细节，保持核心冲突与反转不变。

### 第2步：排版
格式化剧本，添加场景标题、说话人标识、动作和情绪描述。

### 第3步：分镜
生成每15秒4-5个分镜提示词，详细描述镜头类型、人物动作、表情、情绪。

### 第4步：分镜优化
按视频制作模板格式化分镜，添加场景描述和人物标记。

### 第5步：定妆照+场景
生成人物定妆照和场景概念图的 AI 绘画提示词。

### 第6步：关键帧
生成关键帧图片的 AI 绘画提示词，用于视频制作。

## 技术实现

- **前端**：纯 HTML + CSS + JavaScript
- **API**：OpenAI Chat Completions API
- **存储**：LocalStorage 保存设置和历史
- **设计**：现代深色主题，响应式布局
- **字体**：Inter 字体，Font Awesome 图标

## 提示词模板

所有提示词模板都定义在 `prompts.js` 中，可根据需要修改：

- `REWRITE_PROMPT` - 洗稿提示词
- `FORMAT_PROMPT` - 排版提示词  
- `STORYBOARD_PROMPT` - 分镜提示词
- `STORYBOARD_TEMPLATE_PROMPT` - 分镜优化提示词
- `CHARACTER_PROMPT` - 人物定妆照提示词
- `KEYFRAME_PROMPT` - 关键帧提示词

## 注意事项

1. **API 成本**：使用 GPT-4 模型可能产生较高费用，建议先用 GPT-3.5 Turbo 测试
2. **网络连接**：需要稳定的网络连接访问 OpenAI API
3. **内容审核**：生成的剧本内容应符合平台规范
4. **本地存储**：API Key 仅存储在浏览器本地，不会上传到服务器

## 故障排除

### API 连接失败
1. 检查 API Key 是否正确
2. 确保网络可以访问 `api.openai.com`
3. 检查账户余额和额度

### 生成内容不理想
1. 调整"创造力"参数（0-1之间）
2. 尝试不同模型（GPT-4o 效果最好）
3. 增加"最大长度"参数

### 界面问题
1. 清除浏览器缓存
2. 检查控制台错误信息
3. 尝试其他浏览器
