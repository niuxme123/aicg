使用说明

  1. 打开 index.html 在浏览器中
  2. 在右侧设置面板输入 OpenAI API Key
  3. 在左侧输入框中粘贴原始剧本
  4. 点击"一键生成"开始全流程
  5. 或点击步骤导航栏切换到特定步骤
  6. 查看中间区域的不同标签页查看结果
  7. 使用右上角按钮复制或下载结果

核心功能

prompts.js - 提示词模板

包含6个主要提示词模板：
- REWRITE_PROMPT: 洗稿
- FORMAT_PROMPT: 排版  
- STORYBOARD_PROMPT: 分镜
- STORYBOARD_TEMPLATE_PROMPT: 分镜优化
- CHARACTER_PROMPT: 人物定妆照
- KEYFRAME_PROMPT: 关键帧

app.js - 核心功能

主要函数：
- callOpenAI(): 调用 OpenAI API
- runRewriteStep(): 洗稿
- runFormatStep(): 排版
- runStoryboardStep(): 分镜
- runStoryboardOptStep(): 分镜优化
- runCharacterStep(): 人物定妆照
- runKeyframeStep(): 关键帧
- runAllSteps(): 一键生成
- runCurrentStep(): 单步执行

下一步操作

  • 1. 打开 主页面
  • 2. 输入 OpenAI API Key
  • 3. 点击"测试连接"验证 API
  • 4. 点击"加载示例"获取测试剧本
  • 5. 点击"一键生成"开始处理
